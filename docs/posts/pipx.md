# pipx

[https://pipx.pypa.io/stable/](https://pipx.pypa.io/stable/)

前置需求：Python 3.8+

Linux 安裝：

```
python3 -m pip install --user pipx
python3 -m pipx ensurepath
```

* 安裝至 `~/.local/bin/pipx`

macOS 安裝：

```
brew install pipx
pipx ensurepath
```

* 安裝至 `/opt/homebrew/bin/pipx`
